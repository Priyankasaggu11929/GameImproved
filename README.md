# quizgame
Quiz Game in Phaser

# Installation
## Pre requisites
 * NodeJS (https://nodejs.org/en/download/package-manager/)
 

## Install the Quiz Game
```sh
git clone https://github.com/alanblins/quizgame.git
cd quizgame
npm i
```

## Run the game
```sh
npm start
```
open in your browser 
```
http://localhost:8080
```


* Then install again:
```sh
npm i
```
